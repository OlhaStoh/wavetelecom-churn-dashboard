# 📊 WaveTelecom — Churn Analysis Dashboard

An interactive Tableau dashboard analyzing customer churn 
behavior for a telecom company.

## 🔗 Live Dashboard
[View on Tableau Public](https://public.tableau.com/app/profile/olha.stohnushenko/viz/Telecom_Churn_Analysis_OS/General)

## 📌 Overview
Section 1: Churn Overview & Geographic Analysis
This dashboard explores churn patterns across:
- Geography (Churn Map by US states)
- Contract types & payment methods
- Churn reasons & categories
- Customer service call impact

Section 2: Age Analysis
- Churn Rate by Age Group (10-Year Bins)
- Senior vs. Non-Senior Comparison
- Group Contract Impact on Churn
- Average Age of Churned Customers

Section 3: Payment & Contract Analysis
- Contract Type vs. Churn Rate
- Payment Method vs. Churn Rate
- Cross-Dimension: Contract × Payment Scatter Analysis
- Extra Charges and Churn

Section 4: Data & International Plan Analysis
- International Plan & Activity Matrix
.....


  
## 📈 Key Metrics
| Metric | Value |
|---|---|
| Total Customers | 6,687 |
| Churned Customers | 1,796 |
| Churn Rate | 26.9% |
| Retention Rate | 73.14% |
| Avg Monthly Charge | $31.0 |
| Avg Account Length | 32.3 months |

## 🔍 Key Findings
- **44.8%** of churn is driven by competitor offers
- Month-to-month contracts account for **87.9%** of churned customers
- Customers with **4–5 service calls** churn at nearly 100%
- Direct Debit is the dominant payment method (**71.1%**)

## 🗂️ Dashboard Pages
1. **Churn Overview** — KPIs, map, contracts, payment, reason analysis
2. **Age Analysis** — churn rate by age group
3. **Payment & Contract** — deep dive into billing behavior
4. **Data & Intl** — data plan and international usage impact
